# VisualxSwaroop Portfolio

A modern portfolio website showcasing graphic design, UI/UX, and frontend development projects.

## 🌐 Live Website
https://visualxswaroop.vercel.app

## 🚀 Features

- Modern futuristic UI
- Smooth animations using Framer Motion
- Responsive design for all devices
- Project showcase with hover effects
- Interactive hero section with avatar
- Contact form with email functionality
- Social media integration

## 🛠 Tech Stack

Frontend
- React
- Vite
- Tailwind CSS
- Framer Motion

Icons
- Lucide Icons

Backend / Email
- Vercel Serverless Functions
- Resend Email API

Deployment
- Vercel
